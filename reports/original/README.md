# Archived original reports

These files are retained for audit history and comparison with the original project state. They are not the current source of truth.

- The original predictive-model report is superseded by `../generated/leakage_safe_model.json` because the audit identified target-derived item inference and holdout-selection concerns.
- The original statistical report is reproduced by `../../src/statistical_analysis.py`, which also adds a customer-cluster bootstrap estimate and corrected interpretation rules.
- The visualization workbook contains dashboard-supporting tables and a visualization guide; it does not contain the 15 referenced image files.

Use the reproducible scripts in `src/`, the outputs in `reports/generated/`, and the standalone dashboard in `dashboard/index.html` for the published portfolio narrative.
